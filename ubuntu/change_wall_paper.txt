ubunu下用命令设置壁纸：

    gsettings set org.gnome.desktop.background picture-uri "file:【fileName】"
    eg:
        gsettings set org.gnome.desktop.background picture-uri "file:/home/geekgao/图片/美国队长2.jpg"
        gsettings set org.gnome.desktop.background picture-uri "file:/home/liu/图片/Wallpapers/wallhaven-374083.jpg"
    note:
        can create a shell file to excute this
